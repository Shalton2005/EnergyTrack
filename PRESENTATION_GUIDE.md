## 💬 HANDLING QUESTIONS

### **Common Questions & Answers:**

**Q: How is this different from smart plugs?**  
**A:** Smart plugs only control ON/OFF. We monitor live power, predict bills, detect appliances automatically, and work at the main meter level - covering entire house, not just one socket.

**Q: What if WiFi is down?**  
**A:** Device stores last 1 hour of data in RAM, syncs when WiFi returns. We're adding LoRaWAN in Version 2 for no-WiFi areas.

**Q: Can this work with solar panels?**  
**A:** Yes! V2.0 will support bi-directional monitoring for solar input. Many users have requested this.

**Q: How do you make money if FREE tier exists?**  
**A:** FREE is limited (7 days data, no predictions). Users upgrade for ML features, IoT control, and unlimited history. Our conversion rate target is 15%.

**Q: What about data privacy?**  
**A:** All data encrypted (TLS 1.2). Stored on Indian servers (Bangalore AWS region). Users can delete account anytime (GDPR-compliant).

**Q: Why not use existing ML models?**  
**A:** Global models don't recognize Indian appliances (e.g., immersion heaters, mixer-grinders). We trained on Indian household data for 85%+ accuracy.

**Q: Hardware certification timeline?**  
**A:** CE/FCC in progress (₹2L cost). BIS application submitted. Expected approval in 6 months. Meanwhile, selling as "testing kit" (legal).
